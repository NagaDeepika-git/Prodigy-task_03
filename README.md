# Tic Tac Toe

## Overview

A simple web-based Tic Tac Toe game developed during my Web Development Internship at Prodigy InfoTech.
This project demonstrates the implementation of game logic using JavaScript along with an interactive and responsive user interface.

## Features

* Two-player gameplay(Player X and Player O)
* Win and draw detection
* Reset functionality
* Clean and responsive interface

## Tech Stack

* HTML, CSS, JavaScript

## Usage

1. Clone or download the repository
2. Open the project folder
3. Run `index.html` in any web browser

## Learning Outcomes

* Implemented game logic using JavaScript
* Managed user interactions and events
* Improved problem-solving skills
* Built a responsive and interactive UI
